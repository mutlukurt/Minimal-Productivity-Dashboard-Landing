# Lumen — Minimal Productivity Dashboard (Landing Page)

This repository contains a minimal, responsive, and accessible landing page for "Lumen" — a fictional productivity dashboard focused on clarity and privacy.

Features:
- Minimal, solid-color design system
- Responsive layout with accessible navigation and keyboard-friendly interactions
- Sections: Hero, Features, Highlights, Pricing, Testimonials, FAQ, Contact
- Vanilla HTML, CSS, and JavaScript (no frameworks)
- Accessibility-minded: aria attributes, focus styles, reduced-motion support

Files:
- `index.html` — main page structure
- `styles.css` — design system and layout
- `app.js` — interaction logic (mobile nav, FAQ accordion, pricing toggle, form validation, reveal animations)
- `assets/` — small SVG assets used by the page
- `LICENSE` — MIT License (Mutlu Kurt)

How to preview locally:

```bash
cd /path/to/Minimal-Productivity-Dashboard-Landing
python3 -m http.server 8000
# then open http://localhost:8000
```
